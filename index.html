Idlelands

An idling game set in the realm of IRC. It has a modular backend that can be added to any existing frontend (if the frontend uses node).

Idle Lands

Idle Lands is an idling game that takes a couple twists on the original IdleRPG formula. The primary difference is the amount and breadth of events and items that we support and the comical situations that can arise. The second difference is the combat system.

Want to join?

We're in a couple of irc networks, currently we're in irc.snoonet.org, irc.esper.net, irc.freenode.net -- the channel is #idlebot. You can find us in irc.esper.net##idlebot if you want to chat with the devs.
